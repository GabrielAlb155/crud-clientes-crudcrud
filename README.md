# 📋 CRUD de Clientes com Fetch API

Projeto desenvolvido em HTML, CSS e JavaScript para praticar o consumo de APIs utilizando a Fetch API.

## 📖 Descrição

A aplicação realiza um CRUD básico de clientes utilizando a API do CrudCrud.

É possível:

* Cadastrar clientes.
* Listar todos os clientes cadastrados.
* Excluir clientes.

O projeto foi desenvolvido para praticar requisições HTTP utilizando JavaScript.

## 🚀 Funcionalidades

* Cadastro de clientes (POST)
* Listagem de clientes (GET)
* Exclusão de clientes (DELETE)
* Consumo de API utilizando Fetch API
* Manipulação de dados em formato JSON

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Fetch API
* CrudCrud API

## 📂 Estrutura do Projeto

```text
crud-clientes-crudcrud/

└── README.md
```

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/GabrielAlb155/crud-clientes-crudcrud.git
```

2. Abra a pasta no VS Code.

3. No arquivo `scripts.js`, substitua o token da API pelo seu token do CrudCrud.

4. Execute o projeto utilizando a extensão **Live Server**.
